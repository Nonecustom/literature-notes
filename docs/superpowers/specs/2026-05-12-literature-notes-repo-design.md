# Literature Notes Repository Design

## Goal

Create a GitHub-ready repository for literature notes, starting from the existing Word note file and preserving its learning-oriented structure.

## Repository Shape

- `README.md` provides the main index and links to each research direction.
- `notes/embodied-intelligence/` stores notes for the 具身智能 direction.
- `templates/paper-note-template.md` provides the reusable note structure.

## Note Structure

Each paper note includes basic metadata, a stable paper link section, summary, core points, unresolved problems, value, key figure understanding, core data, and unfamiliar terms.

## Publishing

The repository should be initialized locally, committed, connected to a GitHub remote, and pushed once GitHub access is available on the machine.
