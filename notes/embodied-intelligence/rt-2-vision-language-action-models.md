# RT-2: Vision-Language-Action Models Transfer Web Knowledge to Robotic Control

## 基本信息

- 方向：具身智能
- 年份：2023
- 论文稳定链接：
  - arXiv: https://arxiv.org/abs/2307.15818
  - DOI: https://doi.org/10.48550/arXiv.2307.15818
  - Project Page: https://robotics-transformer2.github.io

## 总结

RT-2 是一种依托大模型基础，将指令与机器人的控制流以文本字符串的形式进行保存和执行，便于大模型推理（链式）。

RT-2 把机器人动作写成文本字符串，让大模型既能理解指令，又能直接输出控制信号。

## 核心点

1. 动作 token 化：连续动作 -> 256 个离散 bin -> 文本字符串
2. 联合微调：Web 图文数据 + 机器人控制数据一起训练
3. 涌现能力：大模型把 Web 学到的常识迁移到物理任务中

## 待解决问题

本论文于 2023 年发布，大模型的推理过程在秒级，而我们需求的反应在 ms 级，因此太慢了。

## 作用

- 理解了大模型 + 机器人的核心范式
- 知道这个领域的瓶颈在哪里（速度）
- 以后看到类似论文，可以用“速度”这个维度去评价

## 关键图理解

图片 + 文本 -> VLM 编码器 -> LLM 推理 -> 动作 token 串 -> 解码 -> 机器人执行

## 核心数据

| 方法 | 未见任务成功率 |
| --- | --- |
| RT-1 | 32% |
| RT-2 (5B) | 56% |
| RT-2 (55B) | 62% |

结论：参数量越大，泛化能力越强。

## 看不懂的术语

- token：把连续值（如移动 2.3 cm）映射成离散编号（如 128 号）
- VLM：视觉语言模型，能同时理解图像和文字
- 涌现能力：没专门学过，但模型自己“悟”出来的能力
